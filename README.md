# To Do List
 
Учебный проект — список задач с drag & drop сортировкой. Написан на чистом JavaScript без библиотек и фреймворков.
 
## Возможности
 
- Добавлять, удалять и отмечать задачи выполненными
- Поиск задачи по названию
- Удаление всех задач сразу
- Сохранение списка в `localStorage` — задачи остаются после перезагрузки страницы
 
## Технологии
 
- Vanilla JavaScript (ES6+)
- CSS
- localStorage
 
## Как запустить
 
Открыть `index.html` в браузере. Сборка не нужна.
 
## Что практиковал
 
- `localStorage` — сохранение и загрузка данных между сессиями
- Работа с DOM — динамический рендер списка через `innerHTML`
- События `click` и `change` через делегирование на родительском элементе
- Управление состоянием внутри класса
